系统解析结果

主配送中心: 容迪亚伊 (Jundiaí)

数学计算是严谨的。通过将坐标放置在笛卡尔平面上，圣保罗、伊图和安帕鲁之间的几何精确平衡点（重心）落在了容迪亚伊 (Jundiaí)市。

/ 数学验算

三角形的重心是其顶点坐标的算术平均值。

> Y 轴 (纬度):
Y = (Y1 + Y2 + Y3) / 3
Y = (-23.55 + -23.26 + -22.70) / 3
Y = -69.51 / 3 = -23.17
> X 轴 (经度):
X = (X1 + X2 + X3) / 3
X = (-46.63 + -47.29 + -46.76) / 3
X = -140.68 / 3 = -46.89
