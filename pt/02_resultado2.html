CD Master: Wuhan

A lógica matemática escala para qualquer número de polos. O centroide de um polígono é a média das coordenadas de seus n vértices. Neste caso, o centro logístico exato cai próximo a Wuhan, conhecida como a "Chicago da China" devido ao seu papel como principal hub de transportes do país.

/ Gabarito Matemático

Para n=4 polos com demandas iguais, dividimos a soma das coordenadas por 4.

> EIXO Y (Latitude):
Y = (Y1 + Y2 + Y3 + Y4) / 4
Y = (39.90 + 31.23 + 23.12 + 29.56) / 4
Y = 123.81 / 4 = 30.95
> EIXO X (Longitude):
X = (X1 + X2 + X3 + X4) / 4
X = (116.40 + 121.47 + 113.26 + 106.55) / 4
X = 457.68 / 4 = 114.42
